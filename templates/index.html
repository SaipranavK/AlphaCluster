NASA SpaceApps Challenge 2018

In a nutshell

Unified platform for space data
References: NASA SpaceApps Challenge: Can you build (Do you know when next rocket launch is ?)

Functionalities

User can browse information about past launches, organisations assoicted to respective countries, popular spaceports and launch sites.

Developed with

Flask (microframework for Python based on Werkzeug, Jinja 2) - http://flask.pocoo.org/
SqlAlchemy (Python SQL toolkit and Object Relational Mapper ) - https://www.sqlalchemy.org/
Bootstrap (open source toolkit for developing with HTML, CSS, and JS) - https://getbootstrap.com/

Local Execution

Mount folder directory in CMD and enter the following commands.
set FLASK_APP=AlphaCluster.py
python -m flask run
Copy the local host ip http://127.0.0.1:5000/ and paste in your preferred browser

Future Work

All the organisations can store their data on this platform to have transparency for public user and global communication between their peers.
